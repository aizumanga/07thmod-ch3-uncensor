# Unofficial Higurashi Chapter 3 Original-Art Variant

> **Credit and status:** The installer infrastructure and original mod are the
> work of the [07th-Mod team](https://github.com/07th-mod). This unofficial
> fork only replaces the Tatarigoroshi full script package. This version was
> **vibe-coded with substantial AI assistance** and is not endorsed by
> 07th-Mod.

## Installation

- Download the installer to your computer before running it.
- Windows users should normally choose
  `Higurashi.Ch3.Variant.Installer.Windows.exe`.
- Linux and macOS builds are provided in the same release.
- General Higurashi installation requirements remain those documented by the
  [07th-Mod wiki](https://07th-mod.com/wiki/Higurashi/Higurashi-Getting-started/).
- Report problems specific to this variant in the
  [fork repository](https://github.com/aizumanga/07thmod-ch3-uncensor/issues),
  not to the 07th-Mod team.

## Existing 07th-Mod installations

Users who already installed Chapter 3 with 07th-Mod may download
`Tatarigoroshi.Voice.and.Graphics.Patch.zip` and extract it directly into the
Tatarigoroshi game directory, allowing existing files to be overwritten. A
full reinstall is not required.

## Scope

The alternate calls are limited to censorship level 0 plus Original/Ryukishi
art. All other art and censorship combinations retain the upstream behavior.
The three included custom assets are currently clothed placeholders.

## Windows installer variants

1. `Higurashi.Ch3.Variant.Installer.Windows.exe` — default; requires
   administrator privileges.
2. `Higurashi.Ch3.Variant.Installer.Windows.SafeMode.exe` — text-based launcher;
   requires administrator privileges.
3. `Higurashi.Ch3.Variant.Installer.Windows.NoLauncher.zip` — no launcher and
   no administrator requirement; extract it outside Program Files and the game
   directory, then run `install.bat`.
